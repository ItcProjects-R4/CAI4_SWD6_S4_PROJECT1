# CAI4_SWD6_S4_PROJECT1

## Magento2Demo Website Testing Project

### Project Overview

This project focuses on performing comprehensive testing of the **Magento2Demo e-commerce website** using both **Manual Testing** and **Automation Testing** techniques.

The main goal of this project is to ensure the website functions correctly, identify hidden defects, and improve the overall user experience by detecting bugs in both major functionalities and small details.

---

## Project Objectives

* Validate the core functionalities of the website.
* Detect functional and UI issues.
* Ensure a smooth user experience.
* Apply professional software testing practices using manual and automated approaches.

---

## Technologies & Tools Used

### Manual Testing

* Google Sheets (for test cases, bug reports, and test documentation)

### Automation Testing

* Java
* Selenium WebDriver

## How to Run the Automation Scripts

1. Extract the **Team1 Final Project** file.
2. Open **IntelliJ IDEA**.
3. Click **Open Project**.
4. Select the **AutomationScript** folder.
5. Navigate to:

`src → test → java → tests`

6. Open the **Action Test** file.
7. Run the desired test scripts.
